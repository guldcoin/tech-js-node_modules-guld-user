Guld user management tools.

### Install

##### Node

```sh
npm i guld-user
```

### Usage

##### Node

```
const { getName, getFullName, validate, exists, getHosts } = require('guld-user')
```

##### CLI

See [guld-user-cli](https://github.com/isysd/tech-js-node_modules-guld-user-cli).
